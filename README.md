# Employee Management System (Angular)

A simple Employee Management System built using Angular. This project helps manage employee data with full CRUD operations.

---

## Features
- Add new employee
- View employee list
- Edit employee details
- Delete employee
- Data stored using JSON/Local Storage (or API if  used)
- Responsive UI

---

## Tech Stack
- Angular 21
- TypeScript
- HTML, CSS
- RxJS
- JSON / LocalStorage (or API)
